# Unity Gear VR

Simple Project for Gear VR, CardBoard, etc.


## Prerequires

1. Git 2.6+
2. Unity 5+


## How to Play

Clone

```
git clone https://github.com/humbertodias/unity-gear-vr.git
```

Run

Open project with Unity and play the **DemoScene.scene**
